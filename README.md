## How I'm working now

Why serverless plus AI? When generation is cheap, infrastructure that scales from zero is real leverage. One person can throw a lot of spaghetti and only pay for what sticks.

I'm building production-grade serverless software on AWS — DynamoDB, Lambda, AppSync, EventBridge, Cognito — with the discipline around it that makes agent-assisted development trustworthy: skills, hooks, automated gates, and review workflows that replicate the work of a real engineering team. (Bonus: the agents happily show up for retro.)

I lean on a number of [Yan Cui's](https://theburningmonk.com) patterns, particularly testing with ephemeral environments and the deployment patterns from his Production-Ready Serverless course.
